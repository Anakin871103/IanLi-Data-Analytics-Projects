# [IanLi-Data-Analytics-Projects] Ian Li's Data Analytics Projects

Welcome to the Ian Li Data Analytics Projects repository! This collection showcases various data analysis projects, including exploratory data analysis, machine learning models, and SQL analysis.

## Project Structure

The project is organized as follows:

- **README.md**: Overview, installation instructions, and usage guidelines for the project.
- **.gitignore**: Lists files and directories that should be ignored by Git, such as virtual environments and compiled artifacts.
- **LICENSE**: Contains licensing information for the project.
- **requirements.txt**: Lists the required Python packages and their versions for the project.

### Data

- **data/**: Contains both raw and processed data.
  - **raw/**: Stores original data files in formats such as .csv, .xlsx, and .json.
  - **processed/**: Stores processed data files in formats such as .csv and .parquet.

### Notebooks

- **notebooks/**: Contains Jupyter notebooks for various analyses.
  - **01_Exploratory_Data_Analysis/**: Notebooks for exploratory data analysis.
    - `EDA_Project_A.ipynb`: Notebook for Exploratory Data Analysis Project A.
    - `EDA_Project_B.ipynb`: Notebook for Exploratory Data Analysis Project B.
  - **02_Machine_Learning_Models/**: Notebooks for machine learning models.
    - `Model_Project_C_DecisionTree.ipynb`: Notebook for Decision Tree Model Project C.
    - `Model_Project_D_RandomForest.ipynb`: Notebook for Random Forest Model Project D.
    - `Model_Project_E_XGBoost.ipynb`: Notebook for XGBoost Model Project E.
    - `Model_Project_F_NeuralNetwork.ipynb`: Notebook for Neural Network Model Project F.
  - **03_SQL_Analysis/**: Notebooks for SQL analysis.
    - `SQL_Queries_for_Database_X.ipynb`: Notebook for SQL queries for Database X.
    - `SQL_Case_Study_Y.sql`: SQL case study Y file.

### Source Code

- **src/**: Contains source code for the projects.
  - **utils/**: Contains utility functions.
    - `data_preprocessing.py`: Python script for data preprocessing.
    - `model_training.py`: Python script for model training.
  - **models/**: Contains model definitions.
    - `custom_model_class.py`: Python script for custom model classes (if applicable).
  - **scripts/**: Contains various scripts.
    - `etl_script_for_project_G.py`: ETL script for Project G.
    - `database_interaction.py`: Python script for interacting with the database.

### Projects

- **projects/**: Contains individual projects.
  - **Project_A_Customer_Churn_Prediction/**: Customer churn prediction project.
    - `README.md`: Overview and description of the project.
    - **notebook/**: Contains Jupyter notebooks for the project.
      - `Customer_Churn_Analysis.ipynb`: Notebook for customer churn analysis.
    - **src/**: Contains source code for the project.
      - `churn_model.py`: Python script for the customer churn model.
    - **data/**: Contains data for the project.
      - `customer_data.csv`: CSV file with customer data.
  - **Project_B_Sales_Forecasting/**: Sales forecasting project.
    - `README.md`: Overview and description of the project.
    - **notebook/**: Contains Jupyter notebooks for the project.
      - `Sales_Forecasting_with_XGBoost.ipynb`: Notebook for sales forecasting using XGBoost.
    - **src/**: Contains source code for the project.
      - `sales_etl.py`: ETL script for sales data.
    - **data/**: Contains data for the project.
      - `sales_data.csv`: CSV file with sales data.
  - **SQL_Case_Study_C_E-commerce_Analytics/**: SQL case study for e-commerce analytics.
    - `README.md`: Overview and description of the project.
    - **sql/**: Contains SQL query files.
      - `e_commerce_queries.sql`: SQL file for e-commerce queries.
      - `database_schema.sql`: SQL file for database schema (optional).

### Archived Projects

- **archived_projects/**: Contains older projects that are not currently highlighted (optional).

## Installation

To get started with this project, clone the repository and install the required packages:

```bash
git clone https://github.com/yourusername/IanLi-Data-Analytics-Projects.git
cd IanLi-Data-Analytics-Projects
pip install -r requirements.txt
```
