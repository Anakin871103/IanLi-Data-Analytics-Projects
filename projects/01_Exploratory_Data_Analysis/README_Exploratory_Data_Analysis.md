# 📊 電商客戶行為探索性分析 (E-commerce Customer Behavior EDA)

---

## 🚀 專案概述

本專案旨在對一家虛擬電商平台的客戶購買數據進行深入的探索性資料分析 (EDA)。透過數據清洗、豐富的視覺化以及統計假設檢定，我們旨在揭示客戶的行為模式、偏好，並找出潛在的商業機會。這份分析將為後續的客戶分群、精準行銷策略制定以及銷售預測提供堅實的數據基礎。

**🔗 Jupyter Notebook**: [ECommerce_Customer_EDA.ipynb](./notebook/ECommerce_Customer_EDA.ipynb)

---

## 🎯 專案目標

1.  **理解客戶結構：** 分析客戶的人口統計特徵（年齡、性別、地區）分佈。
2.  **洞察購買行為：** 探索購買頻率、平均訂單價值、最受歡迎的產品類別。
3.  **時間趨勢分析：** 識別銷售額和訂單量的時間性模式（每日、每週、每月）。
4.  **商品銷售熱點：** 找出最暢銷和滯銷的商品。
5.  **高價值客戶識別：** 初步定義和識別潛在高價值客戶。
6.  **統計假設驗證：** 針對某些觀察到的現象進行統計顯著性檢定（例如：新用戶與老用戶的平均消費是否存在顯著差異）。

## 🗃️ 資料來源與描述

本分析使用了從 [數據集來源，例如 Kaggle: E-commerce Dataset for Customer Analysis](https://www.kaggle.com/datasets/example_dataset_link) 獲取的模擬電商交易數據。

**數據集主要包含以下欄位：**
* `CustomerID`: 客戶唯一識別碼
* `Age`: 客戶年齡
* `Gender`: 客戶性別
* `Region`: 客戶所在地區
* `ProductCategory`: 購買的產品類別
* `PurchaseAmount`: 購買金額
* `OrderDate`: 訂單日期
* `IsNewCustomer`: 是否為新客戶 (布林值)
