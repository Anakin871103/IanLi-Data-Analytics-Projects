# SQL Case Study: E-commerce Analytics

## Overview
This project focuses on analyzing e-commerce data to derive insights and improve business strategies. It includes SQL queries that help in understanding customer behavior, sales trends, and inventory management.

## Installation
To set up the project, ensure you have the following prerequisites:

1. Python 3.x installed on your machine.
2. A SQL database (e.g., MySQL, PostgreSQL) where you can run the provided SQL scripts.

### Steps to Install
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/IanLi-Data-Analytics-Projects.git
   ```
2. Navigate to the project directory:
   ```
   cd IanLi-Data-Analytics-Projects/projects/SQL_Case_Study_C_E-commerce_Analytics
   ```
3. Install the required Python packages:
   ```
   pip install -r ../../requirements.txt
   ```

## Usage
1. **SQL Queries**: 
   - Open the `e_commerce_queries.sql` file to view the SQL queries used for analysis.
   - Execute the queries in your SQL database to retrieve the necessary data.

2. **Database Schema**: 
   - The `database_schema.sql` file contains the schema for the database. You can use it to set up your database structure.

3. **Analysis**: 
   - Use the insights gained from the SQL queries to inform business decisions and strategies.
